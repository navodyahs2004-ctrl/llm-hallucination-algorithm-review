# LLM Hallucination Algorithm Review

This repository is created to study, analyze, and review hallucination detection algorithms in large language models.

## Repository Structure

- papers/ : research papers
- notes/ : paper notes and explanations
- pseudocode/ : extracted algorithm steps
- src/ : Python code
- results/ : outputs and observations
- docs/ : final review writing
